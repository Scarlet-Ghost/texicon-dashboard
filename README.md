# Texicon Executive Dashboard

Streamlit dashboard for the Texicon Group (agricultural chemicals, Philippines). Presents revenue, cash-flow, operations, customer-reconnection, and sales-intelligence views over quarterly Excel exports.

- **Entry point:** [`dashboard/app.py`](dashboard/app.py)
- **Full documentation:** [`dashboard/README.md`](dashboard/README.md)
- **Local run:** `cd dashboard && python3.11 -m streamlit run app.py`
- **Live deploy:** Streamlit Community Cloud — see deploy section in `dashboard/README.md`.
